== Description ==

ImageTight is a professional-grade image optimization plugin designed for speed, SEO, and developer performance. Unlike many other plugins that send your private images to unknown servers, ImageTight utilizes a high-performance **Vercel Edge API** to process your images securely and efficiently.

Created by a developer who needed to shrink a 600-page book app from **500MB down to 132MB** to pass Google Play's strict publishing limits, ImageTight is now available as a free tool for every WordPress site.

### Key Features:

*   **Massive Reductions:** Achieves 70-80% compression with zero visible quality loss.
*   **WebP Conversion:** Automatically converts heavy PNGs and JPGs to high-performance WebP.
*   **Restore Original:** Every optimized image keeps a hidden backup. Restore the original uncompressed file anytime with one click.
*   **SEO Alt-Text Editor:** Edit your SEO titles and Alt-tags directly inside the optimization dashboard.
*   **Real-Time Analytics:** Beautiful dashboard showing exact storage saved and library optimization percentage.
*   **Developer Friendly:** Built on a modern Edge architecture for maximum reliability.

== Installation ==

1. Upload the `imagetight-wp-plugin` folder to the `/wp-content/plugins/` directory, or upload the .zip through the WordPress admin.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the **ImageTight Pro** tab in your sidebar.
4. Paste your free API key from [https://imagetight.tabaix.com](https://imagetight.tabaix.com).
5. Run your first library scan and start saving space!

== Frequently Asked Questions ==

= Is there a free tier? =
Yes! Every account starts with 100 free image compressions per month on our Edge network.

= Can I restore my original images? =
Absolutely. ImageTight creates a `.itc-bak` backup of your original file before optimization. You can undo any compression from the "Restore" tab.

= Does WebP work on all browsers? =
Yes. All modern browsers (Chrome, Safari, Firefox, Edge) now support WebP natively.
